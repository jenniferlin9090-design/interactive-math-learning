1. Definition

A quadratic function is a polynomial function with degree 2. Its standard form is:

f(x) = ax2 + bx + c
Here, a, b, and c are real numbers and a ≠ 0.

Other useful forms include:

  • Vertex form: f(x) = a(x - h)2 + k
  • Intercept form: f(x) = a(x - p)(x - q)

2. Key Features

  • The graph is called a parabola.
  • If a > 0, the parabola opens upward.
  • If a < 0, the parabola opens downward.
  • The vertex is the turning point of the parabola.
  • The axis of symmetry passes vertically through the vertex.

3. Examples

  • f(x) = x2
  • f(x) = 2x2 - 3x + 1
  • f(x) = -x2 + 4x - 2

Each example has a highest power of 2, so each one is quadratic.

4. How to Read the Graph

  • The yellow point marks the vertex.
  • The blue line marks the axis of symmetry.
  • The green point marks the y-intercept.
  • The red curve shows the current function.
