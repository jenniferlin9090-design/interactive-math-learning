Activities

Activity 1: Graph Detective for Linear Equations

Concept: Linear equations in two variables, ax + by = c.

Objective: Students identify how a, b, and c affect a straight-line graph, including slope, x-intercept, y-intercept, and solution points.

Student Tasks:

  • Open the interactive linear equation lesson.
  • Set a = 2, b = 3, and c = 6. Record the slope and intercepts shown on the page.
  • Move only c and describe how the line changes position.
  • Move a and b separately, then explain how the slope changes.
  • Choose one point on the line and substitute its x- and y-values into ax + by = c to prove it is a solution.

Expected Product: A short explanation with one sketch, one verified solution point, and one sentence explaining the meaning of slope or intercepts.

Activity 2: Parabola Explorer for Quadratic Functions

Concept: Quadratic functions, f(x) = ax2 + bx + c.

Objective: Students investigate how a, b, and c affect the shape and position of a parabola, including opening direction, width, vertex, axis of symmetry, and y-intercept.

Student Tasks:

  • Open the interactive quadratic function lesson.
  • Set a = 1, b = 0, and c = 0. Record the vertex, axis of symmetry, and y-intercept.
  • Change a to a negative value and describe what happens to the opening direction.
  • Change b and observe how the vertex and axis of symmetry move.
  • Change c and explain how the y-intercept and graph position change.

Expected Product: A completed observation table comparing three different quadratic functions and a short paragraph explaining which coefficient caused the biggest visible change.

Common Misconceptions

Linear Equations in Two Variables

Common Misconceptions:

  • Mistake: Every equation with x and y is linear.
    Correct idea: It is linear only when x and y have degree 1, such as ax + by = c.
  • Mistake: One solution is the only answer.
    Correct idea: A linear equation in two variables has many solution points on the same straight line.
  • Mistake: Slope and y-intercept mean the same thing.
    Correct idea: Slope tells steepness; y-intercept is where the line crosses the y-axis.
  • Mistake: Changing c changes the steepness of the line.
    Correct idea: Changing c usually shifts the line; a and b affect the slope.
  • Mistake: A vertical line is not a linear graph.
    Correct idea: If b = 0 and a is not 0, the graph is a vertical line.

Quadratic Functions

Common Misconceptions:

  • Mistake: All curved graphs are quadratic.
    Correct idea: A quadratic function has degree 2 and can be written as f(x) = ax2 + bx + c.
  • Mistake: The vertex is always at (0, 0).
    Correct idea: The vertex changes when b and c change.
  • Mistake: c controls the width of the parabola.
    Correct idea: c moves the graph up or down; a controls opening direction and width.
  • Mistake: A negative a-value means there is no graph.
    Correct idea: A negative a-value makes the parabola open downward.
  • Mistake: The axis of symmetry is always the y-axis.
    Correct idea: The axis of symmetry passes through the vertex and is not always x = 0.

User Manual

This guide helps students and teachers use the website effectively for learning linear equations in two variables and quadratic functions.

Introduction to the Product

My Mathematics Learning Website is an interactive learning resource for exploring graphs, equations, activities, misconceptions, and assessments. It supports visual learning through sliders, guided questions, images, and Kahoot! assessment links.

Learning Objectives

  • Understand linear equations in two variables in the form ax + by = c.
  • Interpret slope, intercepts, and solution points from a straight-line graph.
  • Understand quadratic functions in the form f(x) = ax2 + bx + c.
  • Identify vertex, axis of symmetry, opening direction, and y-intercept of a parabola.

Step-by-Step Instructions

  1. Start from the Concepts section and choose a topic.
  2. Use the sliders in the interactive lesson to observe graph changes.
  3. Read the guided questions and record observations.
  4. Complete the related activity in the Activities section.
  5. Review Common Misconceptions to correct mistakes.
  6. Scan the QR code or open the Kahoot! link in Assessments.

Target Users

  • Students: Lower to middle secondary students learning algebra and graph interpretation.
  • Teachers: Mathematics teachers who need visual demonstrations, guided activities, and quick formative assessments.
  • Classroom use: Suitable for individual exploration, pair work, or teacher-led explanation.

Expected Learning Outcomes

  • Students can connect equations with their graph shapes.
  • Students can explain how coefficients affect graphs.
  • Students can identify and correct common misconceptions.
  • Students can answer topic-based Kahoot! questions with better confidence.

Limitations or Constraints

  • The interactive graphs are designed for learning and visualization, not advanced graphing precision.
  • Kahoot! links require internet access and may require students to enter a game challenge page.
  • Very small screens may require scrolling to view graphs, activities, and QR codes comfortably.
  • The website focuses only on selected concepts: linear equations in two variables and quadratic functions.
