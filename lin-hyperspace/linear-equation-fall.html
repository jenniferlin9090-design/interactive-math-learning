1. Definition

A linear equation in two variables is an equation that can be written with x and y, where both variables have degree 1.

ax + by = c
Here, a, b, and c are real numbers,
and a and b cannot both be 0.

Every solution is an ordered pair (x, y) that makes the equation true.

2. Key Features

  • The graph is always a straight line.
  • Most lines can be written as y = mx + k.
  • The slope tells how steep the line is.
  • The x-intercept is where the line crosses the x-axis.
  • The y-intercept is where the line crosses the y-axis.

3. Examples

  • 2x + 3y = 6
  • 10x + 4y = 3
  • -x + 5y = 2

These are linear equations in two variables because x and y are both to the first power.

4. How to Read the Graph

  • The red line shows all solutions to the equation.
  • The green point marks the x-intercept when it exists.
  • The yellow point marks the y-intercept when it exists.
  • Any point on the line is a solution.
